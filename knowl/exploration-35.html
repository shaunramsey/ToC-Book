Exploration 16.

The most difficult part of proving that any CFL is context free using the Pumping Lemma, is finding the best string to use as your example. Consider the following language:
\begin{equation*} L_3 = \{a^nb^nc^md^me^m\} \end{equation*}
What is a string that we can use to prove \(L\) is a CFL using the Pumping Lemma?
Answer.
\(s^{(1)} = c^pd^pe^p\)
Was that string unexpected? Were you expecting something closer to:
\begin{equation*} s^{(1)} = a^pb^pc^pd^pe^p \end{equation*}
That string will not work for the Pumping Lemma for CFLs.
Another example of this phenomrnon occures when trying to choose a string to pump from the following language:
\begin{equation*} L_4 = \{a^ib^jc^k | i=j \mbox{ and } j\leq k\} \end{equation*}
The string, \(s^{(1)}=a^pb^pc^p\) will not allow us to show that \(L_4\) is not CFL as using a singular \(c\) as the \(y\) portion of the pumpable string would be able to be pumped and stay in the language.
