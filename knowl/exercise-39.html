Exercise 4.

Create an FA for the following regular expression.
\begin{equation*} REX_{10} = dd/dd/(201d+202d) \end{equation*}
\begin{equation*} \Sigma(REX_{10})=\{d,/\} \end{equation*}
Here, \(d\) is used to represent any decimal digit.
Answer.
There would typically be states between each character, but in the interest of space, the FA has been compressed.
