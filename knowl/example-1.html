Example 10.1.
Here is an example that is applicable to more general computer science. Imagine parsing a computer program. You have run a REX which returns these terminals:
b boolean expressions
i an \(if\) keyword
c a code statement
e an \(else\) keyword
f an \(elif\) keyword
: a colon
Write the CFG using \(b\text{,}\)\(i\text{,}\)\(c\text{,}\)\(e\text{,}\)\(f\text{,}\) and \(:\) for Python \(if\) statements. You may ignore returns and tabbing.
Answer 1.
\(CFG\text{:}\)
\(S \rightarrow iB:CFE\)
\(B \rightarrow bB | b\)
\(C \rightarrow cC | c\)
\(F \rightarrow fB:C | FF | \epsilon\)
\(E \rightarrow e:C | \epsilon\)
The CDFG can also be written as seen below.
Answer 2.
\(CFG\text{:}\)
\(S \rightarrow ib:U_1U_2U_3 | \epsilon\)
\(U_1 \rightarrow cU_1 | c\)
\(U_2 \rightarrow fB:U_1U_2 | \epsilon\)
\(U_3 \rightarrow e:U_1 | \epsilon\)
