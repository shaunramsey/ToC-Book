Consider how the rooms are labeled using the counting numbers:
\begin{equation*} \mbox{Counting Numbers} = \{1,2,3,4,...\} \end{equation*}
and the patrons currently residing in the rooms correspond to the natural numbers:
\begin{equation*} \mathbb{N} = \{0,1,2,3,...\} \end{equation*}
What set would this make with the infinite amount of bus patrons joining the hotel? What would this look like in from a "folding" pattern? The bus patrons are numbered -1, -2, and onwards. This is the set of integers.
\begin{equation*} \mathbb{Z} = \{0,-1,1,-2,2,-3,3,...\} \end{equation*}
There is a one-to-one correspondance among the sets.
