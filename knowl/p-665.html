There are other speciality types of TMs as well.
  • The Regular\(_{TM}\) is an undecidable machine that accepts when the language of a input machine is regular.
    \begin{equation*} \mbox{Regular}_{TM} = \{\langle M\rangle | L(M) \mbox{ is regular} \} \end{equation*}
  • The Tape\(_{TM}\) is an undecidable machine that writes an "a" to the given tape.
    \begin{equation*} \mbox{Tape}_{TM} = \{\langle M\rangle | M \mbox{ writes and "a" to the tape}\} \end{equation*}
  • The S\(_{TM}\text{.}\)
    \begin{equation*} S_{TM} = \{ \langle M \rangle | M \mbox{ does not accept } M \} \end{equation*}
