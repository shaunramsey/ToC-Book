Construct a CFG for \(L_5\) where
\begin{equation*} L_5 = \{a^i(bc)^i | i \geq 0 \} \end{equation*}
Before looking at the correct grammar, letâ€™s first analyze some common incorrect answers. \(CFG_5'\) below demonstrates one common mistake:
\(CFG_5'\text{:}\)
\(S \rightarrow ASB | \epsilon\)
\(A \rightarrow aA | \epsilon\)
\(B \rightarrow bcB | \epsilon\)
But, why is \(CFG_5'\) incorrect? Construct some strings using this grammar.
  • abc
  • aabcbc
  • aabcbc
  • aaaaaaa
But, how did the string \(abcbc\) come about from \(CFG_5'\text{?}\)
\(S \rightarrow ASB\) \(S\) is replaced by \(ASB\)
\(ASB \rightarrow aSB\) The \(A\) is replaced by \(a\)
\(aSB \rightarrow aASBB\) The \(S\) is replaced by \(ASB\)
\(aASBB \rightarrow aaSBB\) The \(A\) is replaced by \(a\)
\(aaSBB \rightarrow aaBB\) The \(S\) is replaced by \(\epsilon\)
\(aaBB \rightarrow aabcB\) The \(B\) is replaced by \(bc\)
\(aabcB \rightarrow aabc\) The \(B\) is replaced by \(\epsilon\)
By not forcing the \(a\) and \(bc\) to be created at the same time and inculding an epsilon, then it is possible to a string to be created that is not in the language.
