Exploration 15.

Consider the following language:
\begin{equation*} L_2 = \{a^ib^jc^k | \mbox{ where } i = j \mbox{ and } j \mbox{ less than } k \} \end{equation*}
We will now prove that \(L_2\) is not context free.
Proof.
Assume that \(L_2\) is a CFL. \(L_2\) can be rewritten as:
\begin{equation*} L_2 = \{a^ib^ic^k | \mbox{ where } i \mbox{ less than } k \} \end{equation*}
  1. Consider \(s^{(1)} = a^Pb^Pc^{P+1}\) where \(s^{(1)} \in L_2\text{.}\)
  2. Since \(s^{(1)}\) is of length at least \(p\text{,}\) it should obey the Pumping Lemma for Context Free Languages.
    • Consider a section of \(s^{(1)}\text{,}\) a section made up of the last \(a\) and the first two \(b\)’s. By pumping this section up by one, a new string, \(s^{(2)} = a^{P+1}b^{P+1}c^{P+1}\) is created, but this string is no longer a part of \(L_2\text{.}\)
    • Consider another section of \(s^{(1)}\text{,}\) one that is made up of the last \(b\) and first two \(c\)’s. Pumping this string down by one creates the new string, \(s^{(0)} = a^Pb^{P-1}c^P\text{,}\) but this string is no longer in \(L_2\text{.}\)
  4. Therefore, by contradiction, \(L_2\) is not a CFL.
