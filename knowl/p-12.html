The intersection of two sets is a new set that contains all the things in common from the two sets. Typically, the symbol \(\cap\) is used as a binary operator like: \(A \cap B\text{.}\) If a thing is in both sets, it is in the intersection of the two sets. It is possible for the intersection to be empty \(\emptyset\) or to contain all of one set (when one set is the subset of another). Let’s do an example. Let’s take set
\begin{equation*} D = \{a,b,y,ab,ba,by,ay,bay,abba,baby\} \end{equation*}
and set
\begin{equation*} E = \{w | w\ \text{is any English word}\} \text{.} \end{equation*}
Their intersection would have to be both an English word and one of the words inside the set \(D\text{.}\) Assuming we know which words are actually English words the intersection would be:
\begin{equation*} D \cap E = \{a,by,bay,baby\}\text{.} \end{equation*}
As food for thought, when would the intersection of two sets be \(\emptyset\text{?}\)
