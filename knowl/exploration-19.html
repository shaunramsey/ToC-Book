Exploration 3.

Let us consider a grammar with multiple variables.
\(CFG_4\text{:}\)
\(S \rightarrow aS | T | \epsilon\)
\(T \rightarrow bT | \epsilon\)
Can we generate only \(a\)s?
\(S \rightarrow aS\) \(S\) is replaced by \(aS\)
\(aS \rightarrow aaS\) \(S\) is replaced by \(aS\)
\(aaS \rightarrow aa\epsilon\) \(S\) is replaced by \(\epsilon\)
The resulting string is \(aa\text{.}\) Therefore, \(CFG_4\) can generate only \(a\)s.
Can we generate only \(b\)s? How?
Answer.
\(S \rightarrow T\) \(S\) is replaced by \(T\)
\(T \rightarrow bT\) \(T\) is replaced by \(T\)
\(bT \rightarrow bbT\) \(T\) is replaced by \(bT\)
\(bbT \rightarrow bb\epsilon\) \(T\) is replaced by \(\epsilon\)
The resulting string is \(bb\text{.}\) Therefore, \(CFG_4\) can generate only \(b\)s.
What are some other strings that we can generate using \(CFG_4\text{?}\)
  • \(\displaystyle ab\)
  • \(\displaystyle aaabb\)
  • \(\displaystyle abbbbbb\)
The language of this grammar is the empty string or any amount of \(a\)s followed by any amount of \(b\)s.
