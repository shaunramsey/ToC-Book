As some sets are infinite, we will often use set-builder notation to describe sets. An example of set-builder notation to describe a set is:
\begin{equation*} \{w | w\ \text{is a list of letters that contains an}\ a\}\text{.} \end{equation*}
This should be read as the set of all \(w\)’s where \(w\) is a string (list of letters) that contains an \(a\text{.}\) We will see this in again in an example shortly. This symbol \(|\) is often called a pipe or line. Some programmers may think of it as the ’or’ symbol. However, in this notation, the pipe, \(| \text{,}\) is read as where or such that. You can think of it as a declaration or assignment if you like. This is a collection of things defined on the left, where all the things follow the rules on the right.
