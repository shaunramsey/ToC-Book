Exercise 2.

The maximum size of the power set is \(2^n\) where \(n\) is the number of states in the machine. Where does this come from?
Hint.
There is always a choice to be in a state or not.
Answer.
There are two choices with each state, whether to be in it or to not be in it. This means that there are two options for each state, making the maximum size of the power set \(2^n\text{.}\)
