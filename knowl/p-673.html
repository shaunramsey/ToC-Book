The Acceptance Turing Machine is a set of pairs of machines and inputs where the machine accepts the input. In other words:
\begin{equation*} A_{TM} = \{ \langle M,w \rangle | M \mbox{ is a TM and } M \mbox{ accepts } w \} \end{equation*}
Here, you get the input and break it into the machine,\(M\text{,}\) and the input,\(w\text{.}\) Then, you simulate \(M\) as a TM running on \(w\text{.}\) If \(M\) accepts, then the \(A_{TM}\) accepts. If the \(M\) rejects, then the \(A_{TM}\) rejects.
