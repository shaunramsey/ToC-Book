Paragraph

NFAs make it possible to exist in two states at once. You might think of this as parallelism. Parallelism can occur because a state has multiple paths for the same character. It can also happen because of an empty string transition. Empty string transitions are free jumps to another state. Sometimes these multiple paths and free jumps are called guesses when discussing FAs. Only one execution path must lead to an accept, to accept a string in these conditions.
