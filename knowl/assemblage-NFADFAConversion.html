Exploration 2. Getting from a NFA to a DFA.

For this, consider \(M_{12}\text{.}\)
\(M_{12}\) is a NFA. An \(1\) at \(A\) will lead to both \(B\) and \(D\text{.}\) This is an example of parallelism. What is the maximum number of states a DFA created from \(M_{12}\) could have?
Answer 1.
\(32\text{.}\) A DFA made from \(M_{12}\) can have up to \(32\) states as \(M_{12}\) has \(5\) states and \(2^5 = 32\text{.}\)
How is a NFA made into a DFA? The goal is to make a set of all posible destination states for each character from each state. In \(M_{12}\text{,}\) a \(1\) at \(A\) could go to \(B\) or \(D\text{.}\) A \(0\) at \(A\) would go to an \(\emptyset\) state.
Try to create the states that \({B,D}\) would lead to. There is a posibility that each state in a set is active. If one state in a set is an accept state, then the state that includes it in its set is also an accept state.
