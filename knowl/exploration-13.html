Exploration 13. RLs Closed under Intersection.

Intersection \(\cup\) of \(L_1\cup L_2\) is any string that is accepted in \(L_1\) and \(L_2\text{.}\) It is very useful in closure proofs and in crafting NFAs for languages that use the logic of \(and\text{.}\) For example look at these two machines:
\begin{equation*} L_1(M_1) = \{\mbox{all strings that end with a 1}\} \end{equation*}
\begin{equation*} L_2(M_2) = \{\mbox{all strings with an even number of symbols}\} \end{equation*}
\begin{equation*} L_1\cup L_2 = \{\mbox{is any string that ends in a 1 and that has an even number of symbols.}\} \end{equation*}
Try traceing the path through both machines and only accept if both machines accept. In a more systematic way, consider the states \(Q_1\times Q_2\) and create those machines.
Hint.
Here are some examples of the step through thinking.
  • States with \(B_2\) occur when the string has even length.
  • \(A_2\) is when it is odd
  • \(A_1\) says the last seen symbol was a 0
  • \(B_1\) is when it was a 1
So, it turns out we can use this to build DFAs of challenging languages!
