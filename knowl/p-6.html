Sets are a collection of objects where there is no ordering and elements are not repeated or counted. Sets can contain other sets or they can be empty. Things get really tricky when you think about a set that contains an empty set, but we will not need to deal with that just yet. Typically we use curly brackets to denote set notation (enumeration notation). Thus \(\{a,b,c\}\) is a set that contains elements called \(a\text{,}\) \(b\) and \(c\text{.}\) That is the same set as \(\{b,a,c\}\text{.}\) Ordering does not matter. If a \(d\) were added to this set, we would have a new set \(\{a,b,c,d\}\text{.}\) If an \(a\) were added to this set, it would not change the set because the set already contained an \(a\text{.}\) We will use enumeration notation right away to start describing important elements for this book.
