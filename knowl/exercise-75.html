Exercise 4.

Create a CFG for the language below:
\begin{equation*} L = \{\mbox{all palindromes with } \sigma=\{0,1\}\} \end{equation*}
(Palindromes are strings that the same no matter if read backwards for forwards. \(1011101\) is an example.)
Answer.
\(CFG:\)
\(S \rightarrow 0S0 | 1S1 | 0 | 1 | \epsilon\)
