In order to check if the language can be represented by a TM, we need to establish a line of thinking. We ultimately need to check if there is a certain number of 0s, followed by the same number of 1s. But how do we check for this when heads of TMs can only move right, left, or stay in place? And how can this be represented with the "read move write" structure? For this language, a potential strategy follows:
  1. Mark off an unmarked 0
  2. Move to the first unmarked 1 and mark it off
  3. Repeat steps 1 and 2 until no more 0s are left, also make sure all 1s are marked off too
