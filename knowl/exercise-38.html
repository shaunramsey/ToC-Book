Exercise 3.

Create an FA for the following Regular Expression.
\begin{equation*} REX_9 = (www.\lambda\lambda^*.edu)(//\lambda\lambda^*)^* \end{equation*}
\begin{equation*} \Sigma(REX_9)=\{\lambda,.,//\} \end{equation*}
Here, \(\lambda\) is used to represent any alphabetic symbol.
Answer.
There would typically by states between each character, but in the interest of space, the FA has been compressed.
