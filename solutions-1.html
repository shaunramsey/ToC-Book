    18 Reducibility
    18.3 Exercises

    18.3.1.

    Hint.
    \(Q(x): \)
    • If \(x != w\) then \(Q\) rejects
    • \(M\) accepts \(w : Q\) rejects
    • else \(Q\) accepts \(x\text{.}\)
    \begin{equation*} L(Q) = \emptyset , Q \in E_{TM} \mbox{ when }M \mbox{ accepts } w \end{equation*}
    \begin{equation*} L(Q) = \{x\}, Q \notin E_{TM} \mbox{ when }M \mbox{ does not accept }w \end{equation*}

    18.3.2.

    Hint.
    General idea: Build a machine, \(Q\text{,}\) such that when we pass it to \(R\) (the decider for the language in question), \(R\) accepts \(Q\) when \(M\) accepts \(w\) (and doesn’t when it doesn’t).
