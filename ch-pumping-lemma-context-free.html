    Section 13.1 Revisiting Parse Trees and Spotting Loops

    In order to understand the Pumping Lemma for Context Free Languages, we need to take a step back and look at some parse trees.

    Exploration 13.

    Consider this CFL below:
    \(CFG_1:\)
    \(S \rightarrow aSb | ab\)
    Try creating the parse tree for producing the string \(aaabbb\) from this grammar.
    Answer 1.
    That string was able to be generated by a relatively simple parse tree, but the grammar is not in CNF. Try putting the grammar in CNF.
    Answer 2.
    \(CFG_1:\)
    \(S_0 \rightarrow NB | AB\)
    \(S \rightarrow NB | AB\)
    \(N \rightarrow AS\)
    \(A \rightarrow a\)
    \(B \rightarrow b\)
    Considering the grammar in CNF, re-make the parse tree that would produce the string \(aaabbb\text{.}\)
    Answer 3.
    Can you see how the same inner \(S \rightarrow A\) and \(B\) exists in both parse trees? If that is removed from the middle of the trees, then an \(S \rightarrow S\) loop remains. This loop works from \(S \rightarrow NB \rightarrow ASB \rightarrow AABB\text{.}\) This loop can be removed, or it can be taken as many times as we like.

    Section 13.2 The Pumping Lemma for Context Free Languages

    The Pumping Lemma for Context Free Languages can be used to prove that a language is not context free. These are the conditions of the Pumping Lemma for Context Free Languages:
    • \(\displaystyle |vy| > 0\)
    • \(\displaystyle uv^ixy^iz \in L\)
    • \(\displaystyle |vxy| \leq p\)
    And the most general proof outline follows below:
    1. Assume the language is a CFL
    2. Pick a string \(s^{(1)}\)
    3. Try to pump all possible loops and show that none work (stay in the language)
    4. State that the assumption was wrong and \(L\) is not a CFL

    Exploration 14.

    The outline of the proof that \(L_1\) is not a CFL where
    \begin{equation*} L_1 = a^nb^nc^n \end{equation*}
    is below:
    1. Assume the language, \(L\text{,}\) is a CFL and it obeys the Pumping Lemma for Context Free Languages.
    2. A string of length at least \(p\) will obey the Pumping Lemma for Context Free Languages. Choose to consider \(s^{(1)}\text{,}\) where \(s^{(1)} = a^pb^pc^p\) and \(s^{(1)} \in L\)
    3. When pumping up, \(a\text{,}\) \(b\text{,}\) and \(c\) must each increase to stay in \(L\text{.}\) But, because the pumpable part has to be within \(p\) symbols, there is no way to include all three. (The \(c\)’s are too far away from the \(a\)’s)
    4. Thus, our assumpiton was wrong, \(L\) must not be a CFL by contradiction.
    Let’s look at another example.

    Exploration 15.

    Consider the following language:
    \begin{equation*} L_2 = \{a^ib^jc^k | \mbox{ where } i = j \mbox{ and } j \mbox{ less than } k \} \end{equation*}
    We will now prove that \(L_2\) is not context free.
    Proof.
    Assume that \(L_2\) is a CFL. \(L_2\) can be rewritten as:
    \begin{equation*} L_2 = \{a^ib^ic^k | \mbox{ where } i \mbox{ less than } k \} \end{equation*}
    1. Consider \(s^{(1)} = a^Pb^Pc^{P+1}\) where \(s^{(1)} \in L_2\text{.}\)
    2. Since \(s^{(1)}\) is of length at least \(p\text{,}\) it should obey the Pumping Lemma for Context Free Languages.
      • Consider a section of \(s^{(1)}\text{,}\) a section made up of the last \(a\) and the first two \(b\)’s. By pumping this section up by one, a new string, \(s^{(2)} = a^{P+1}b^{P+1}c^{P+1}\) is created, but this string is no longer a part of \(L_2\text{.}\)
      • Consider another section of \(s^{(1)}\text{,}\) one that is made up of the last \(b\) and first two \(c\)’s. Pumping this string down by one creates the new string, \(s^{(0)} = a^Pb^{P-1}c^P\text{,}\) but this string is no longer in \(L_2\text{.}\)
    4. Therefore, by contradiction, \(L_2\) is not a CFL.

    Exploration 16.

    The most difficult part of proving that any CFL is context free using the Pumping Lemma, is finding the best string to use as your example. Consider the following language:
    \begin{equation*} L_3 = \{a^nb^nc^md^me^m\} \end{equation*}
    What is a string that we can use to prove \(L\) is a CFL using the Pumping Lemma?
    Answer.
    \(s^{(1)} = c^pd^pe^p\)
    Was that string unexpected? Were you expecting something closer to:
    \begin{equation*} s^{(1)} = a^pb^pc^pd^pe^p \end{equation*}
    That string will not work for the Pumping Lemma for CFLs.
    Another example of this phenomrnon occures when trying to choose a string to pump from the following language:
    \begin{equation*} L_4 = \{a^ib^jc^k | i=j \mbox{ and } j\leq k\} \end{equation*}
    The string, \(s^{(1)}=a^pb^pc^p\) will not allow us to show that \(L_4\) is not CFL as using a singular \(c\) as the \(y\) portion of the pumpable string would be able to be pumped and stay in the language.
